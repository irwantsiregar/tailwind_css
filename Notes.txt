## https://tailwindcss.com/

Apa itu Tailwind? 

"Rapidly build modern websites without ever leaving your HTML."

"A utility-first CSS framework packed with classes" like flex, pt-4, text-center and rotate-90 that can be composed to build any design, directly in your markup.

"Bangun situs web modern dengan cepat tanpa harus meninggalkan HTML Anda."

"Kerangka kerja CSS yang mengutamakan utilitas yang dikemas dengan kelas" seperti flex, pt-4, text-center dan rotate-90 yang dapat disusun untuk membangun desain apa pun, langsung di markup Anda.

Tailwind CSS:
- Dibuat oleh Adam Wathan, 1 November 2017
- Saat ini sudah versi 3.0.24
- 55.6K++ Star di Github
- https://github.com/tailwindlabs/tailwindcss


Utilities:
-Preflight
-Layout
-Flexbox & Grid
-Spacing & Sizing
-Typography
-Background & borders
-Effect & Filters
-Transition, Trasformation & Animation 
-Tables
-... dll

Features:
- Interactivity
- Responsive Design
- Dark Mode
- Reusability
- Custom Styles
- Functions & Directives

Software Requirements

- Code Editor: Visual Studio Code
- VSCode Extension:
	- Tailwind CSS InteliSense
	- Live Priview
	- PostCSS Language Support
- Package Manager: NPM(Node.js)
- Terminal: PowerShell, Git Bash
